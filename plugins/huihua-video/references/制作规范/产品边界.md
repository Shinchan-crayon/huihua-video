# 产品边界

## 产品身份

`huihua-video` 是把文章或口播制作成手绘绘本式动态视频的独立 Codex 插件。`Shin-video` 是另一套本地口播与视频生产系统。两者不是同一个产品的不同名称、版本、轻量模式或兼容模式。

## 独立命名空间

| 项目 | huihua-video |
| --- | --- |
| 公开入口 | `$huihua-video` |
| 内部 Skill | `$huihua-*` |
| 用户配置 | `~/.config/huihua-video` |
| 中间产物 | `<project>/.huihua-video-runtime/` |
| Remotion 模板 | 插件自带 `Remotion绘画视频模板` |

## 强制隔离

- 不调用任何 `$shin-video-*` Skill。
- 不读取或修改名为 `Shin-video` 的仓库目录。
- 不读写 `.shin-video-runtime`。
- 不复用 Shin-video 的配置、模板、场景清单、缓存、旧图片、旧音频或渲染产物。
- 项目素材必须位于当前 huihua-video 项目目录内；不接受 `shin-video://` 或跨项目绝对路径。
- 不创建或依赖工作流状态台账。

两个产品可以同时安装，但一次任务只能进入其中一条工作流。用户明确调用 `$huihua-video` 时，只执行本插件的总控与 `$huihua-*` 内部 Skill。
